# 🤖 AI HR 智能招聘系统

AI驱动的全栈智能人力资源招聘管理系统，基于 Next.js + React + Tailwind CSS 构建。

## ✨ 核心功能

### 📊 仪表盘
- 实时数据概览：岗位数、候选人总数、今日新增、今日面试
- 筛选数据统计：通过率、平均分、待处理 Offer
- 最新动态流、快捷操作入口

### 💼 岗位管理
- 发布/编辑/关闭岗位
- 候选人关联统计
- 状态筛选（招聘中/已关闭/草稿）

### 👥 候选人管理
- **简历批量采集**：从招聘网站自动获取候选人信息，关联在招岗位建人才库
- **智能筛选评分**：按分数自动分通过/待定/淘汰，从硬性条件（学历、经验、技能、薪资）和个人素养打分，可调权重，核验学历/薪资
- **AI面试题库**：破冰、自我介绍、离职动机、情景模拟等题型，每道题配优中差回答参考
- 候选人详情面板：完整档案、技能标签、评分维度详情

### 🎤 面试管理
- **面试分析评估**：录音转录、笔记记录，AI和人工一起评估能力、性格、情绪需求
- **录用谈判辅助**：面试通过后纳入录用名单，AI给薪资谈判建议和风险预判
- **面试官复盘优化**：生成复盘建议，分析面试官优势和改进方向

### ⚙️ 系统设置
- 评分权重配置（学历、经验、技能、薪资、潜力）
- 筛选阈值设置（通过线、淘汰线）
- 功能开关（自动筛选、AI辅助决策）

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📦 技术栈

- **框架**: Next.js 14 (App Router)
- **UI**: React 18 + Tailwind CSS
- **语言**: TypeScript
- **部署**: GitHub Pages (静态导出)

## 🌐 部署到 GitHub Pages

项目已配置 GitHub Actions 自动部署。推送到 `main` 分支后会自动构建并部署到 GitHub Pages。

1. 在 GitHub 仓库设置中启用 Pages
2. Source 选择 "GitHub Actions"
3. 推送代码即可自动部署
